    Research Gap
    • Specially designed to meet the needs of visually impaired students, Guided object pointing and identification system for the camera convenient for the VI students.
    • Implement Pre proved descriptive object learning systems to efficiently support object identification and learning, specially designed for elementary level and primary level vision-impaired students.
    • Customizable voice-based input-output system for the convenient use of the functionality for VI learners.
    • Especially, these activities are aimed at identifying objects for visually impaired children and helping them learn. By creating these activities, it is possible to enhance the child's object recognition skills that they have learned. This increases the child's creativity and ability to learn independently. Thus, it represents a novel approach to improving the object identification abilities of children with primary visual impairment through engaging activities.
    Introduction
    • Vision-impaired individuals face many challenges mentally as well as educationally when working in the real world.
    • Identifying and learning about objects in the environment can be highly advantageous for visually impaired students during the beginning of education.
    • Specialized training is required for individuals who can assist in providing knowledge about object identification and teaching.
    • Regularly checking the progress of visually impaired (VI) students is crucial for their educational development and overall well-being.
    Research Problem
    primary research problem is to develop an intelligent voice assistive system to enhance object identification and comprehensive learning for elementary and primary vision-impaired students through a mobile application. This system aims to provide detailed and descriptive answers about objects such as fruits, vegetables, animals, vehicles, and more, specifically tailored for the needs of visually impaired students. Through consultations with teachers and students at schools for the visually impaired, and by analyzing existing datasets and literature, we identified the necessity for an effective tool that can provide real-time object recognition and detailed audio descriptions. Additionally, the system needs to incorporate interactive activities and games to facilitate learning and engagement, track student progress, and offer personalized suggestions to improve learning outcomes.
    Research Objectives

    Enable vision-impaired elementary-level students to identify simple objects using mobile cameras and receive descriptive guided learning about those objects by voice output.

    Specific Objectives
    • Capture the object using camera helping Guided object pointing and identification system
    • Identify the object using computer vision algorithms
    • Generate a comprehensive learning description according to the object
    • Deliver the generated output as a voice output
    Methodology

    The methodology for enhancing object identification and comprehensive learning for elementary and primary vision-impaired students via a mobile application involves several key stages: data collection, system design, implementation, and evaluation. Each stage is essential to ensure the effectiveness and usability of the final application.
